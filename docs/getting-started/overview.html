Overview

Pushpin is a static site generator focused on simplicity. There are tons of great static site generators like Zola, Jekyll, Hugo, and mdBook.

Quick tour

We'll walk through creating a new site, building it, and examining its structure. Then we'll add a new page and template.

Create and examine a new site

By adding a directory, we have created a Section.

Write some content in example-page.md. Associate it with its template with Frontmatter:

---
template: example-page.html
---

# Example page header

This is an example page.

Add this code for the template:

<!DOCTYPE html>
<html>
  <head></head>
  <body>
  {{ content | safe}}
  </body>
</html>

The Markdown content is compiled to HTML, and then inserted into templates via content. | safe escapes the string for reasons. Check out the Templates page for more information.

Finally, navigate to http://127.0.0.1:7878/example-section/example-page.html to see the generated page.
